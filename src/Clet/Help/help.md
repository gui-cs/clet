## Examples

```sh
# Show the main help overview (TUI viewer):
clet help

# Render to stdout instead of the TUI:
clet help --cat

# Show help for a specific clet:
clet help select
clet help range
clet help md

# Equivalent forms — all work:
clet select help
clet select --help
clet select -h

# Render a clet's help to stdout:
clet help select --cat
clet select help --cat
```
